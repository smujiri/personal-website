    Initialize a new Astro project equipped with Tailwind CSS and TypeScript.

Phase 2: Multilingual Infrastructure

    The consultant speaks English, Georgian, Russian, German, and Spanish. Set up Astro's routing to support subdirectories for each language (e.g., /en/, /ka/, /es/).

    Implement a sleek language switcher in the main navigation header.
